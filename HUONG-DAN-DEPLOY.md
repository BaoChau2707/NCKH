## 🌐 CÁCH 1: NETLIFY (DỄ NHẤT - KHUYẾN NGHỊ)

### Bước 1: Vào Netlify
```
https://netlify.com
```
Click **Sign up** → Chọn **GitHub** (hoặc Email)

### Bước 2: Deploy
1. Vào https://app.netlify.com/drop
2. **Kéo thả** toàn bộ folder `study-garden` vào
3. Đợi 10-30 giây
4. **XONG!** 🎉

### Bước 3: Lấy link
```
https://random-name-123.netlify.app
```

### Bước 4: Đổi tên (Optional)
1. Click vào site
2. **Site settings** → **Change site name**
3. Đổi thành: `study-garden`
4. Link mới: `https://study-garden.netlify.app`

### ✅ Ưu điểm:
- Siêu dễ (kéo thả là xong)
- Nhanh (10 giây)
- Tự động HTTPS
- Miễn phí mãi mãi

## 🌐 CÁCH 2: GITHUB PAGES (PHỔ BIẾN)

### Bước 1: Tạo GitHub account
```
https://github.com/signup
```

### Bước 2: Tạo repository mới
1. Vào https://github.com/new
2. Tên repo: `study-garden`
3. Chọn **Public**
4. Click **Create repository**

### Bước 3: Upload code

**Cách A: Dùng GitHub Desktop (Dễ)**
1. Tải: https://desktop.github.com/
2. Cài đặt và đăng nhập
3. **File** → **Clone repository** → Chọn `study-garden`
4. Copy toàn bộ file vào folder vừa clone
5. Viết commit message: "Initial commit"
6. Click **Commit to main**
7. Click **Push origin**

**Cách B: Upload trực tiếp trên web**
1. Vào repo vừa tạo
2. Click **uploading an existing file**
3. Kéo thả toàn bộ file
4. Click **Commit changes**

### Bước 4: Bật GitHub Pages
1. Vào repo → **Settings**
2. Sidebar → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → **/ (root)**
5. Click **Save**

### Bước 5: Đợi và truy cập
Đợi 1-2 phút, vào:
```
https://USERNAME.github.io/study-garden/
```

Thay `USERNAME` bằng tên GitHub của bạn.

### ✅ Ưu điểm:
- Phổ biến, nhiều người dùng
- Tích hợp với GitHub
- Miễn phí
- Dễ update (push code mới)
